## 大纲
#### 每周周记

1. 记录这周看到的有启发的文章
2. 记录这周所看的书，看了那些、有什么感想和收获
3. 记录这周所听的书、课程及其他--主要来自得道
4. 另外，就是总结这周的工作，有那些值得注意的地方

## 目的

1. 梳理自己在成长的过程中的收获，提炼整理自己的想法和思考。
2. 作为别人认识自己的一个窗口，而不是为了宣传和盈利，和别人交朋友的态度来阐述自己的思想。
3. 认识世界的一种方式方法。
